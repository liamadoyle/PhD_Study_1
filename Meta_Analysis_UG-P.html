Project Description

This is an R Markdown file that documents for some of the analyses conducted for the first study in my dissertation. Specifically, this file documents the multi-level meta-analytic model conducted for the effect of psychopathy on behaviour in the Ultimatum Game as a Proposer (UG-P).
